# ☀️ Weather App Project

A simple weather app that fetches and displays current weather data for any city using the WeatherAPI.

## Features
- Search weather by city name
- Displays current temperature in Fahrenheit
- Realtime weather data from WeatherAPI
- Clean, responsive UI
- Error handling for invalid cities

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- WeatherAPI

## Setup Instructions
1. Clone the repository
```bash
git clone <your-repo-url>
cd weather-app
```

2. Get your API key
   - Sign up at [WeatherAPI.com](https://www.weatherapi.com/)
   - Copy your API key

3. Add your API key
   - Open `test.js`
   - Replace the `API_KEY` value with your actual API key

4. Run the application
   - Open `index.html` in your browser
   - Or use a local server (e.g., Live Server in VS Code)
